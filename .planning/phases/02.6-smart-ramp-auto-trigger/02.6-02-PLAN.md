# Pipeline Runner: Both InMail + Static per Cohort + Image-Local Fallback (SR-03, SR-04)

## Objective

Refactor the existing `main.py --ramp-id` pipeline to (a) produce BOTH InMail and Static campaigns for every cohort (today it picks one arm per row), (b) gracefully save the generated PNG locally when LinkedIn `create_image_ad` is blocked by the LINKEDIN_MEMBER_URN issue (today it raises and aborts), and (c) expose `run_launch_for_ramp(ramp_id, modes)` as a clean programmatic entry point for Plan 01's poller.

**Purpose:** Phase 2.6 needs the poller to drive the existing pipeline machinery WITHOUT subprocess wrapping (Pattern 4 — in-process call gives clean exception handling and lets Plan 01 classify `consecutive_failures`). Two new behaviors are required: dual-arm dispatch per cohort, and a 403 fallback that doesn't kill the run. Both lock cleanly into the existing dispatch site at `main.py:519` and the existing exception handler at `main.py:721-740`.

**Output:**
- `main.py` — extract `_resolve_cohorts()` from `_process_row` lines 222-517; add `_process_static_campaigns()` mirroring `_process_inmail_campaigns()`; new `run_launch_for_ramp(ramp_id, modes, dry_run)` and `_process_row_both_modes()`; new `_save_creative_locally()` helper.

**Step 2 — Locate `create_image_ad(...)` (around lines 392-500). Refactor the function so:**

The existing happy-path body (DSC post creation → creative attach → return creative_urn) is moved INTO an inner `_create_image_ad_impl(...)` that may raise. The public `create_image_ad(...)` wraps that call and translates exceptions into `ImageAdResult`:

```python
def create_image_ad(
    self,
    account_id: str,
    campaign_id: str,
    image_urn: str,
    ad_name: str,
    dsc_text: str,
    dsc_link: str,
) -> ImageAdResult:
    """Create an image-ad creative on LinkedIn.

    Returns ImageAdResult — NEVER raises for the LINKEDIN_MEMBER_URN /
    403 cases (those return status="local_fallback" so callers can fall back
    to saving the PNG locally). Other unexpected errors return status="error".

    Backwards compat: callers that previously bound the result as a string
    MUST migrate to `.creative_urn`. See main.py for the migration.
    """
    try:
        urn = self._create_image_ad_impl(
            account_id=account_id, campaign_id=campaign_id,
            image_urn=image_urn, ad_name=ad_name,
            dsc_text=dsc_text, dsc_link=dsc_link,
        )
        return ImageAdResult(creative_urn=urn, status="ok")
    except RuntimeError as exc:
        msg = str(exc)
        if "LINKEDIN_MEMBER_URN" in msg:
            return ImageAdResult(
                status="local_fallback",
                error_class="RuntimeError",
                error_message=msg,
            )
        return ImageAdResult(
            status="error",
            error_class="RuntimeError",
            error_message=msg,
        )
    except Exception as exc:
        msg = str(exc)
        upper = msg.upper()
        if "403" in msg or "FORBIDDEN" in upper:
            return ImageAdResult(
                status="local_fallback",
                error_class=type(exc).__name__,
                error_message=msg,
            )
        return ImageAdResult(
            status="error",
            error_class=type(exc).__name__,
            error_message=msg,
        )
```

The inner `_create_image_ad_impl(...)` keeps the existing logic verbatim (LINKEDIN_MEMBER_URN check + DSC post + creative attach + return urn).

def _process_row_both_modes(row, *, ramp_id, dry_run=False, modes=("inmail","static"), **kwargs):
    """Phase 2.6: run cohort discovery ONCE, then dispatch BOTH InMail + Static.

    Per-cohort isolation: each campaign-creation arm wraps its inner loop in
    try/except so one cohort's failure never aborts the others.
    """
    resolved = _resolve_cohorts(row, dry_run=dry_run, **kwargs)
    if not resolved.selected:
        log.warning("No cohorts resolved — skipping both arms")
        return {"ok": True, "campaign_groups": [], "inmail_campaigns": [],
                "static_campaigns": [], "creative_paths": {}, "per_cohort": []}

    inmail_result = {"campaigns": [], "creative_paths": {}}
    static_result = {"campaigns": [], "creative_paths": {}}

    if "inmail" in modes:
        try:
            inmail_result = _process_inmail_campaigns(
                resolved.selected,
                group_name=resolved.group_name,
                project_id=resolved.project_id,
                tg_cat=resolved.tg_cat,
                exclude_pairs=resolved.exclude_pairs,
                ramp_id=ramp_id,
                dry_run=dry_run,
                **kwargs,
            ) or inmail_result
        except Exception:
            log.exception("InMail arm aborted — Static arm will still run")

    if "static" in modes:
        try:
            static_result = _process_static_campaigns(
                resolved.selected,
                group_name=resolved.group_name,
                project_id=resolved.project_id,
                tg_cat=resolved.tg_cat,
                exclude_pairs=resolved.exclude_pairs,
                ramp_id=ramp_id,
                dry_run=dry_run,
                **kwargs,
            ) or static_result
        except Exception:
            log.exception("Static arm aborted — InMail arm result preserved")

    return {
        "ok": True,
        "campaign_groups": [resolved.group_name] if resolved.group_name else [],
        "inmail_campaigns": inmail_result.get("campaigns", []),
        "static_campaigns": static_result.get("campaigns", []),
        "creative_paths": {
            **{f"{k}_inmail": v for k, v in inmail_result.get("creative_paths", {}).items()},
            **{f"{k}_static": v for k, v in static_result.get("creative_paths", {}).items()},

                "cohort_description": getattr(c, "cohort_description", ""),
                "inmail_urn": inmail_result.get("campaigns_by_cohort", {}).get(c.id),
                "static_urn": static_result.get("campaigns_by_cohort", {}).get(c.id),
                "inmail_creative": inmail_result.get("creative_paths", {}).get(c.id),
                "static_creative": static_result.get("creative_paths", {}).get(c.id),
