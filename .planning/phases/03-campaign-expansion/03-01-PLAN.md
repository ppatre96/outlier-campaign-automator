<objective>
Create a targeted, one-shot regeneration script that produces fresh InMail creatives for three existing STEM campaigns (633412886, 635201096, 634012966) using angle F (Financial — rate-in-subject-line), attaches each new creative to its existing campaign URN, and logs the creative URN to Sheets.

Purpose: Angle F is the proven control (is_control=True, CTR/CPA winner in PROJECT.md performance data) but `_process_inmail_campaigns()` rotates through A/B/C by index and never picks F. This plan ships a surgical regen tool that bypasses that gap without creating new campaigns or campaign groups.

Output:
- `scripts/regen_stem_inmail.py` — executable regen entry point with `--dry-run` flag
- README usage snippet
- Three new sponsoredCreative URNs attached to existing campaigns (when run without --dry-run)
- Three new rows in the Sheets "Creatives" tab
</objective>
