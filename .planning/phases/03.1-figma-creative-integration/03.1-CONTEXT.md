## Technical Context

### `build_figma_layered_frame_js()` Function Signature

```python
build_figma_layered_frame_js(
    frame_name: str,           # e.g. "abc123_A_v1"
    headline: str,             # primary text (Inter Bold 86px white)
    subheadline: str,          # secondary text (Inter Regular 46px white)
    angle: str,                # "A", "B", or "C" for gradient colors
    photo_base64: str,         # Gemini PNG as base64 (data:image/png;base64,...)
    earnings: str,             # bottom strip text (default: generic)
) -> str                       # Returns JavaScript for use_figma MCP
```

### Angle-Specific Gradient Colors

- **A**: Light blue (0.68, 0.85, 0.90) + pink (1.0, 0.71, 0.76)
- **B**: Light blue + orange (1.0, 0.82, 0.65)
- **C**: Light blue + light green (0.56, 0.93, 0.56)

### Frame Structure Created

1. 1200×1200 white frame (named per `frame_name` parameter)
2. Background photo (raster, fills frame, from base64 PNG)
3. Gradient A overlay (top-left quadrant emphasis)
4. Gradient B overlay (bottom-right quadrant emphasis)
5. Headline text (y=100, center-aligned, Inter Bold 86px white)
6. Subheadline text (y=853, center-aligned, Inter Regular 46px white)
7. White bottom strip (y=1032, h=168)
8. Earnings text (y=1066, dark brown, earnings text or generic placeholder)

## Pipeline Integration Points

### Current Agent Receives

```python
context = {
    "project_id": str,          # from campaign-manager
    "variants": List[Dict],     # from outlier-copy-writer (3 A/B/C variants)
    "tg_category": str,         # target audience
    # NEW:
    "photo_base64": str,        # Gemini-generated photo as base64
}
```

### Agent Calls

For each variant, call:
```python
js_code = build_figma_layered_frame_js(
    frame_name=f"{project_id}_{variant['angle']}_v1",
    headline=variant["headline"],
    subheadline=variant["subheadline"],
    angle=variant["angle"],
    photo_base64=context["photo_base64"],
    earnings=variant.get("earnings", "generic"),
)
# Then execute via use_figma MCP:
use_figma(fileKey=FIGMA_FILE_KEY, code=js_code, description="Create creative frame")
```
