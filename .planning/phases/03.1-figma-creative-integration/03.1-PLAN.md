## Risks & Mitigations

| Risk | Mitigation |
|------|-----------|
| base64 too large for Figma plugin | Verify plugin accepts 10MB+; if not, add PNG compression before encoding |
| Gradient colors don't match spec | Double-check RGB tuples in `build_figma_layered_frame_js()` against 03.1-CONTEXT spec |
| Text positioning off | Verify y-coordinates (100, 853, 1032, 1066) and font sizes (86px, 46px) match implementation |
| Layers not independently editable | Ensure each layer is a separate frame/group in Figma structure, not merged/flattened |
| Agent instructions unclear | Test-run agent with instructions and confirm it understands input/output contract |

---

## Rollback Plan

If end-to-end test fails:

1. **Photo not embedded:** Check base64 format (must have `data:image/png;base64,` prefix); revert to File Upload path if needed
2. **Gradient colors wrong:** Verify RGB tuples in source code match spec; rerun test with corrected values
3. **Text not visible:** Check font/size/positioning in source; compare against generated frame's actual y/x values
4. **Frame creation fails:** Confirm `use_figma` MCP has write permissions; check Figma file access

**Revert strategy:** Commit checkpoint before Task 2 (pipeline mods); if Task 3 fails, revert Task 2, keep Task 1 (instructions).
