# Phase 3.1: Figma Creative Replication Integration - Summary

**Phase:** 3.1 (inserted between Phase 3 prep and Phase 3 expansion)  
**Duration:** ~90 min (3 tasks, sequential)  
**Status:** Ready for execution  
**Blocker:** None

---

## What We're Doing

Integrating the completed Figma creative layer builder (`build_figma_layered_frame_js()`) into the campaign agent pipeline. The builder creates editable Figma frames with raster photos, gradient overlays, and text — replacing the old text-only clone method.

**Why:** Current pipeline generates images (PNG) but doesn't embed them in editable Figma frames. This phase connects image generation → base64 encoding → Figma frame creation, enabling designers to edit photos, text, and overlays post-generation.

---

## Three Tasks

### Task 1: Update Agent Instructions (15 min)
- **File:** `~/.claude/agents/outlier-creative-generator.md`
- **Change:** Replace Stage 8g to use `build_figma_layered_frame_js()`, document `photo_base64` input
- **Done when:** Instructions clearly reference new function, input/output contract documented

### Task 2: Modify Pipeline (20 min)
- **Files:** `src/midjourney_creative.py` or `scripts/dry_run.py` + `main.py` context assembly
- **Change:** Convert Gemini PNG to base64, add `context["photo_base64"]` before agent call
- **Done when:** No NameError accessing `photo_base64`, conversion works for typical image sizes

### Task 3: End-to-End Test (30 min)
- **File:** `scripts/dry_run.py --flow-id <test-id> --skip-linkedin`
- **Check:** 3 Figma frames created with photo backgrounds, gradient overlays, editable text
- **Done when:** All layers visible, editable, colors correct per angle (A/B/C)

## Risks

| Risk | Impact | Mitigation |
|------|--------|-----------|
| base64 too large | Plugin rejects data | Verify PNG < 10MB; compress if needed |
| Text positioning wrong | Frames unreadable | Compare y-coords (100, 853, 1032, 1066) against implementation |
| Layers not editable | Designers can't customize | Ensure each layer separate in Figma structure |
