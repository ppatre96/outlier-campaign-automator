# Experiment Scientist Agent & Test Directive System

## Objective

Build the middle layer of the feedback loop: consolidate insights from feedback_agent + competitor intelligence, maintain a priority-ordered hypothesis queue, and feed test directives to the creative brief generator at 20/80 split (test/baseline).

**Purpose:** Convert performance insights into systematic A/B testing that guides optimization.

**Output:**
- Experiment backlog (persisted across restarts)
- Test directive JSON consumed by ad-creative-brief-generator
- Hypothesis tracking (which tests succeeded, failed, pending)

**Class: ExperimentBacklog**
```python
class ExperimentBacklog:
    def __init__(self, filepath: str = "data/experiment_backlog.json"):
        self.filepath = filepath
        self.backlog = []  # List[dict] sorted by priority desc
        self.load()
    
    def load(self) -> None:
        """Load backlog from disk; create empty if missing."""
        if Path(self.filepath).exists():
            with open(self.filepath, 'r') as f:
                self.backlog = json.load(f)
                log.info("Loaded %d experiments from backlog", len(self.backlog))
        else:
            log.info("No existing backlog; starting fresh")
            self.backlog = []
    
    def save(self) -> None:
        """Write backlog to disk."""
        Path(self.filepath).parent.mkdir(parents=True, exist_ok=True)
        with open(self.filepath, 'w') as f:
            json.dump(self.backlog, f, indent=2)
        log.info("Saved %d experiments to backlog", len(self.backlog))
    
    def add_hypothesis(self, hypothesis: dict) -> None:
        """
        Add hypothesis to backlog if not already present.
        hypothesis schema: {cohort, angle, photo_subject, reason, expected_impact, priority_score, status}
        status: 'pending' | 'running' | 'completed' | 'failed'
        """
        key = (hypothesis['cohort'], hypothesis['angle'], hypothesis['photo_subject'])
        # Avoid duplicates
        if not any((h['cohort'] == key[0] and h['angle'] == key[1] and h['photo_subject'] == key[2]) 
                   for h in self.backlog):
            hypothesis.setdefault('status', 'pending')
            hypothesis.setdefault('created_at', datetime.utcnow().isoformat())
            self.backlog.append(hypothesis)
            self.backlog.sort(key=lambda h: h.get('priority_score', 0), reverse=True)
            log.info("Added hypothesis: %s/%s", hypothesis['cohort'], hypothesis['angle'])
            self.save()
    
    def peek_next(self, n: int = 1) -> list[dict]:
        """Return top N pending experiments without removing."""
        pending = [h for h in self.backlog if h.get('status') == 'pending']
        return pending[:n]
    
    def mark_running(self, hypothesis_key: tuple) -> None:
        """Mark hypothesis as 'running' (test in progress)."""
        cohort, angle, photo = hypothesis_key
        for h in self.backlog:
            if h['cohort'] == cohort and h['angle'] == angle and h['photo_subject'] == photo:
                h['status'] = 'running'
                h['started_at'] = datetime.utcnow().isoformat()
                self.save()
                break
    
    def mark_completed(self, hypothesis_key: tuple, result: dict) -> None:
        """Mark hypothesis as 'completed' and store result."""
        cohort, angle, photo = hypothesis_key
        for h in self.backlog:
            if h['cohort'] == cohort and h['angle'] == angle and h['photo_subject'] == photo:
                h['status'] = 'completed'
                h['completed_at'] = datetime.utcnow().isoformat()
                h['result'] = result  # {ctr, cpa, conversions, etc.}
                self.save()
                break
    
    def clear_pending(self) -> None:
        """Clear all pending experiments (for reset or testing)."""
        self.backlog = [h for h in self.backlog if h.get('status') != 'pending']
        self.save()
```

**Method 1: ingest_feedback(hypotheses: list[dict], competitor_intel: dict) -> None**
Input:
- hypotheses: from feedback_agent.analyze_creative_performance()
- competitor_intel: loaded from data/competitor_intel/latest.json

Logic:
1. For each hypothesis in list, compute priority_score:
   - base_score = expected_impact (parsed from string "CTR+5%" → 5.0)
   - confidence = len(supporting_evidence) / 3 (capped at 1.0)  [TBD: rule-based confidence scoring]
   - feasibility = 0.9 (assume all are feasible; adjust if blockers detected)
   - priority_score = base_score × confidence × feasibility
2. For each competitor_idea in competitor_intel['experiment_ideas']:
   - If matches existing cohort/angle pattern: boost existing hypothesis score +0.5
   - If new angle: create new hypothesis with competitor_idea as reason, score = 0.7
3. Call backlog.add_hypothesis() for each unique (cohort, angle, photo_subject) tuple
4. Log: "Ingested {N} hypotheses + {M} competitor ideas; backlog now {total} experiments"

**Method 2: generate_test_directive(cohort_name: str) -> dict**
Input: cohort_name (used to filter backlog for this cohort's next test)
Output: test_directive schema: {cohort, angle, photo_subject, priority, expected_impact, test_allocation}

Logic:
1. Call backlog.peek_next(1) filtered to pending + matching cohort_name
2. If no pending experiment for cohort: return {cohort, angle: "baseline", photo_subject: "baseline", test_allocation: 100} (use existing)
3. If pending experiment exists:
   - Return {cohort, angle, photo_subject, priority, expected_impact, test_allocation: 20}
   - Call backlog.mark_running(key) to start the test
4. Log: "Test directive for %s: angle=%s (priority=%.1f)", cohort, angle, priority

**Method 3: get_backlog_status() -> dict**
Returns summary of experiment queue:
```json
{
  "total": 15,
  "pending": 8,
  "running": 4,
  "completed": 3,
  "next_to_test": [
    {cohort, angle, photo_subject, priority_score}
  ]
}
```

- If hypothesis expected_impact not parseable: default to 3.0
- If backlog fails to save: log.error() but return test_directive anyway (in-memory state valid)

**Logging:**

## Test Directive Integration (Phase 2.5 Feedback Loop)

When invoked with a test_directive from experiment_scientist_agent, the brief generator branches:

### Baseline Branch (80% of runs)
- No test_directive provided OR test_allocation = 100
- Use standard prompt: "{cohort_name} is hiring for {role}. Generate a compelling LinkedIn InMail brief."
- Angle defaults to A (financial/rate angle)

### Test Branch (20% of runs)
- test_directive provided with test_allocation = 20
- Extract: test_directive.angle, test_directive.photo_subject
- Modify prompt to emphasize test angle:
  - If angle='B': "Test benefit-focused angle: emphasize flexibility, learning, career growth"
  - If angle='C': "Test community/culture angle: emphasize team, project impact, belonging"
  - If angle='A': "Financial angle: emphasize current tasking rate, reward potential"
- Modify photo_subject instruction:
  - If photo_subject != "baseline": "Primary visual: {photo_subject}. Ensure visual stands out from previous creatives for {cohort_name}."
  - If photo_subject == "baseline": Use standard photo guidance
