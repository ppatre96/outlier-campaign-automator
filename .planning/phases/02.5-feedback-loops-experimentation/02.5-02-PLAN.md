**Class: SlackReactionHandler**
```python
class SlackReactionHandler:
    def __init__(self, slack_bot_token: str, callback_registry: dict = None):
        """
        Initialize handler with Slack bot token.
        callback_registry: dict mapping reaction_emoji -> callback_fn
        """
        self.token = slack_bot_token
        self.client = WebClient(token=self.token)
        self.callbacks = callback_registry or {}
```

**Method: register_reaction_callback(emoji: str, callback: Callable)**
- Stores callback function for a specific emoji
- Example: register_reaction_callback('👍', on_pause_cohort)

**Method: handle_reaction_event(event: dict) -> dict**
Input event schema (from Slack Events API):
```json
{
  "type": "reaction_added",
  "user": "U...",
  "reaction": "thumbsup",  # or "lab"
  "item": {
    "type": "message",
    "channel": "C...",
    "ts": "1234567890.123456"
  }
}
```

Logic:
1. Extract reaction emoji name ("thumbsup" → "👍", "lab" → "🧪")
2. Fetch original message from Slack (client.conversations_history(channel=..., latest=ts, limit=1))
3. Parse cohort_name from message text (look for "cohort_name" or pattern matching)
4. Build action_context: {emoji, cohort_name, user_id, timestamp, message_ts}
5. Look up callback in self.callbacks; if found, call it with action_context
6. Return {success: bool, action: str, cohort: str} result

Emoji mapping:
- "thumbsup" (👍) → action_type = "PAUSE"
- "lab" (🧪) → action_type = "TEST_NEW_ANGLES"

**Method: validate_reaction(emoji: str, cohort_name: str) -> bool**
- Check emoji is 👍 or 🧪 (no other reactions accepted)
- Check cohort_name exists and is non-empty
- Log.warning() and return False if invalid

**Error handling:**
- If message fetch fails: log.warning() and return {success: False, reason: "message_not_found"}
- If cohort_name not parseable from message: log.error() and return {success: False, reason: "cohort_parse_error"}
- If callback throws exception: log.error(), do NOT re-raise
- All errors result in clean failure message, not exceptions to caller

**Logging:**
- log.info("Reaction %s from user %s on cohort %s", emoji, user_id, cohort_name)
- log.debug("Triggering callback for %s", action_type)

**Helper function: parse_cohort_from_message(text: str) -> str | None**
- Regex pattern: r"Cohort ([A-Za-z_0-9]+)" or similar
- Extract cohort name from alert text
- Return cohort_name or None
