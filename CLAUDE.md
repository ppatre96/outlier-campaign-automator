# Outlier Campaign Agent — Claude Guidelines

## Language & Terminology

When generating any copy (ad creatives, headlines, descriptions, CTAs), strictly follow Outlier's approved vocabulary. Never use the words in the "Don't Say" column — always substitute the approved alternative.

| Don't Say | Instead, Say |
|-----------|--------------|
| Required | Strongly encouraged |
| Job | Task, opportunity |
| Role, position | Opportunity |
| Training, growth, learning | Become familiar with project guidelines |
| Project rate | Current tasking rate |
| Bonus | Reward |
| Assign | Match |
| Team | Part of this project / member of this group |
| Instructions | Project guidelines |
| Remove from project | Release from project |
| Discourse | Outlier Community |
| Compensation | Payment |
| Performance | Progress |
| Promote | Eligible to work on review-level tasks |
| Interview | Screening |
| Worker team | You're matched with a project, this project has been prioritized |

This applies to:
- Ad creative headlines and subheadlines (`build_ad_html`, `create_image_ad`)
- LinkedIn campaign names and descriptions
- Any copy written into Figma creatives
- Any user-facing strings generated by the agent
